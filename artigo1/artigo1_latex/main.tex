\begin{abstract}  

O resumo deve ser escrito em um único parágrafo e seguir a ordem lógica: contextualização, objetivo, metodologia, resultados e conclusões. Para usar expressões matemáticas durante o texto use "\$" como exemplificado a seguir. The viscosity of a particular fluid is an interesting parameter that plays an important role in fluid dynamics of that fluid. We chose the common household cooking item canola oil. Using a ball drop, we set out to measure viscosity at various temperatures and create a model for $\eta (T)$ between 0$^{\circ}$C and 100$^{\circ}$C, as well as an accurate measurement for viscosity at room temperature, $\eta (T=20^{\circ}$C). It was found that the viscosity between 0$^{\circ}$C and 40$^{\circ}$C can be approximated using the function $\eta (T) = \alpha e^{-\beta T}$, where $\alpha = 440.65 \pm 30.40 \hspace{0.1cm} mPa\cdot s$ and $\beta = 0.03761 \hspace{0.1cm} \frac{1}{^{\circ}C}$ and that an estimation for viscosity at room temperature is equivalent to $207.6901\pm14.3306 \hspace{0.1cm} mPa\cdot s$. The precision of this measurement was limited by uncertainty in lab equipment used to measure various quantities as well as the image analysis software we used and the limited frame-rate of our camera.

\end{abstract}

%%%%%%%%%%%%%%%%%%%% Começo da Equação 1 %%%%%%%%%%%%%%%%%%%%%%%

\begin{equation}\label{eq1}
    F_{total} = \sum_{i=1}^{} F_{i} = F_{gravity} - F_{buoyant} - F_{stokes' drag}
\end{equation}
%%%%%%%%%%%%%%%%%%%% Fim da Equação 1 %%%%%%%%%%%%%%%%%%%%%%%%%%
Now, making the substitutions\cite{RHK} that $F_{total}=ma$, $F_{gravity}=mg$, $F_{buoyant} = m_{disp}g$\cite{Footnote}, and $F_{stokes drag}=6\pi \eta Rv$, we can create the equation:

%As equações são numeradas automaticamente, para retirar a numeração basta colocar um asterisco no comando \begin{equation}:
%%%%%%%%%%%%%%%%%%%% Começo da Equação 2 %%%%%%%%%%%%%%%%%%%%%%%
\begin{equation*}\label{eq2}
    ma = mg - m_{disp}g - 6\pi \eta Rv
\end{equation*} 
%%%%%%%%%%%%%%%%%%%% Fim da Equação 2 %%%%%%%%%%%%%%%%%%%%%%%%%%

Next we make the substitutions that $\Delta x=\frac{1}{2}at^2$ (and thus $a=\frac{2\Delta x}{t^2}$) and $v=\frac{\Delta x}{t}$.
%%%%%%%%%%%%%%%%%%%% Começo da Equação 3 %%%%%%%%%%%%%%%%%%%%%%%
\begin{equation}\label{eq3}
    m\frac{2\Delta x}{t^2} = mg - m_{disp} - 6\pi \eta R\frac{\Delta x}{t}
\end{equation}
%%%%%%%%%%%%%%%%%%%% Fim da Equação 3 %%%%%%%%%%%%%%%%%%%%%%%

Now, we do some simple algebra to solve for $\eta$, as a function of our parameters $\Delta x$, $t$, $R$, $m$, and $m_{disp}$.

\begin{equation}\label{eq4}
\eta = \frac{mgt}{6\pi R \Delta x} - \frac{m_{disp}gt}{6\pi R \Delta x} - \frac{m}{3\pi Rt}
\end{equation}
